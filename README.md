# Flag Challenge – Canadian Provinces Edition

A modern iOS quiz application built with SwiftUI that tests user's knowledge of Canadian provincial flags.

## 📱 About The Project

Flag Challenge is an interactive quiz app where users identify the flags of Canadian provinces and territories while tracking their best scores.

The app features a clean user interface, smooth navigation, and score tracking functionality.

## 🛠 Built With

- Swift
- SwiftUI
- Xcode

## ✨ Features

- 🇨🇦 Canadian provincial & territorial flag quiz
- 🎯 Multiple-choice questions
- 🏆 High score tracking
- 🔁 Replay functionality
- 🎨 Custom UI design
- 📱 Smooth navigation between screens
